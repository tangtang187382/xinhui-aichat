# 蔚蓝档案同人 AI 聊天平台

> **注意**：  
> - **请使用电脑以获得最佳体验**。  
> - 若使用手机或平板，请**务必横屏**，否则界面显示会《异形》。  
> - **未花模型加载时间较长**（也有可能网速太慢加载不出），加载期间**切勿切换页面或操作其他功能**，否则会导致加载失败
> - **此开源版本所有API需要用自己的，这个版本统一用your-api等代替**，修改api的地方集中在chatview的四个组建，feedbackproblem.vue，home.vue，fund-support.vue，userprofile文件夹下的除了userinformation.vue以外的其他组建，在组建传参和js脚本中。

## 功能特色

- **自定义聊天背景**
- **自定义聊天框颜色**
- **语音对话模式**（在设置中开启）
  - 所有角色均支持语音输出
  - 使用 **阿里云 CosyVoice** 合成
  - **白子的语音最还原**，其他角色相似度可能略低
  - 开启语音后，AI 思考响应时间会有所延长

> 设置路径：进入平台 → 点击「设置」界面 → 调整背景、颜色、语音开关等选项。
